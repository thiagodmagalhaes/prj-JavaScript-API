# prj-JavaScript-API
## O presente projeto foi desenvolvido com o propósito de atender ao desafio imposto em um processo seletivo.

O objetivo central consistiu em consumir uma API e, posteriormente, transformar os dados obtidos em uma interface gráfica. Para tanto, a linguagem de programação JavaScript foi adotada em conjunto com as tecnologias HTML, CSS, Bootstrap e Vue. Ademais, o framework Axios, integrado ao Vue, foi utilizado para o consumo da API.

O projeto em questão proporcionou significativo aprimoramento na compreensão e utilização dos recursos disponibilizados pela linguagem de programação JavaScript. Além disso, evidenciou-se a importância do emprego de ferramentas como o Bootstrap, que se mostrou essencial para a criação do layout, e o Vue, que facilitou de maneira significativa a resolução de problemas específicos, como no caso do consumo da API.
